Online Tool for Anomaly Repair in Time Series

{# Vadetis is web application to perform, compare and validate various anomaly detection algorithms using different configurations. It allows users to upload their own datasets#} {# as well as training data in order to perform outlier detection. The datasets can either be shared with other users or only be used by yourself.#} {# The datasets can be altered by injecting additional outliers.#} {# Look into the FAQ page for more information about the usage of this application.#} {{ Title }} is an online tool to compare various anomaly repair techniques.

{#
Features
#}

Main Parts:

  • Display
    • Real-World Datasets
    • Synthetic Datasets
  • Anomaly Repair
    • Inject and Repair Datasets
    • Compare Repair Techniques
  • Recommendation
    • Model Selection using Flaml
    • Model Selection using RayTune
  • Algorithm Tuning
    • {# Detailed Repair of Matrix#} Matrix decomposition {# Techniques#}
    • Parameter Search {# using Bayesian Optimization or Successive Halving Search#}

Supported algorithms:

  • Iterative Minimum Repairing (IMR) [VLDB 2017]
  • Stream Data Cleaning under Speed Constraints (SCREEN) [SIGMOD 2015]
  • CDrep (Repair based on the Centroid Decomposition) [KAIS 2020]
  • RPCA (Robust PCA using Huber Loss Function) [ARC 17]
    • {# Journal “Avtomatika i Telemekhanika” is published in Russian and translated in ... Abbreviation: АиТ (in Russian), AiT (transliterated), ARC (English title)#}

Algorithm Selection

  • A Fast Library for Automated Machine Learning & Tuning (FLAML)
  • Ray Tune: Hyperparameter Tuning (Paper | Documentation)

This application was implemented at the at the eXascale Infolab, a research group at the University of Fribourg, Switzerland.
